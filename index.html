Tutorial para Baixar o Banco de Base

Nota

Seja bem vindo ao repositório das bases atualizadas do CNES, o intuito da criação desta pagina é facilitar o métodos de download dos arquivos necessários para os computadores referente a secretária, contudo para fazer o download correto dos arquivos siga os passos a seguir:

Requisitos

1-Ter o utorrent instalado em sua maquina :Clique aqui para Baixar

2-Baixar o arquivo da base disponivel aqui:Clique aqui para Baixar

3-Abrir o utorrent e abrir o arquivo no programa segue fotos abaixo explicando:

passo-1 abra o aplicativo utorrent

passo-2 clique no sinal de +

passo-3 escolha o arquivo baixado da base

passo-4 clique nos tres pontos para abrir uma janela de salvamento

passo-5 escolha uma pasta onde deseja salvar o arquivo

passo-6 apos ter selecionado a pasta aperta ok

passo-7 apos o ter baixado completamente o arquivo entre na pasta onde salvou e o procedimento será finalizado
